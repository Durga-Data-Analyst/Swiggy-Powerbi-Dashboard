# Swiggy Data Analysis Dashboard (Power BI)

## 📌 Project Objective
This project analyzes food delivery data from Swiggy to uncover insights related to customer behavior, food preferences, delivery performance, and pricing trends across major Indian cities.

---

## 📊 Dashboard Overview

The interactive Power BI dashboard provides a comprehensive view of:

- Total Orders: 8.68K  
- Total Restaurants: 8.68K  
- Average Ratings: 3.66  
- Average Delivery Time: 53.97 mins  

---

## 📈 Key Features & Visualizations

### 🔹 Food Category Analysis
- Top 5 most served food types
- Indian cuisine dominates with ~28% share
- Comparison across Chinese, Fast Food, South Indian, and North Indian

### 🔹 City-wise Delivery Analysis
- Map visualization showing average delivery time across cities
- Helps identify high and low performance regions

### 🔹 Pricing Insights
- Area-wise average price comparison
- Highlights most expensive locations (e.g., Vile Parle, Brigade Road)

### 🔹 Interactive Filters
- City-based slicer for dynamic filtering
- Enables drill-down analysis by location
